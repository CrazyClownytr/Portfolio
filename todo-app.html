Over het project

De Simple To-Do App is een webapplicatie waarmee gebruikers hun dagelijkse taken kunnen beheren. Gebouwd met HTML, CSS, JavaScript en PHP, laat deze app zien hoe front-end en back-end technologieën samenwerken om een eenvoudige maar krachtige gebruikerservaring te creëren. De applicatie ondersteunt CRUD-functionaliteit (Create, Read, Update, Delete), waarmee gebruikers taken kunnen toevoegen, markeren als voltooid, bewerken en verwijderen.

Wat heb ik gedaan?

  • Ontwerp en implementatie van de gebruikersinterface met HTML en CSS
  • Gebruikersinteractie toegevoegd met JavaScript
  • PHP- en MySQL-backend opgezet voor data-opslag en CRUD-functionaliteit
  • Database ontworpen en gekoppeld via phpMyAdmin
  • Project gedeployed op GitHub als onderdeel van mijn portfolio

Features

  • Taken toevoegen via een eenvoudig invoerformulier
  • Taken markeren als voltooid of verwijderen
  • Automatische opslaan van taken in een MySQL-database
  • Responsief ontwerp dat goed werkt op desktop en mobiel
  • Heldere en gebruiksvriendelijke interface

Galerij

De startpagina van de To-Do App, waar gebruikers hun taken kunnen bekijken.

Gebruikers kunnen eenvoudig een nieuwe taak toevoegen via het formulier bovenaan.

Voltooide taken worden visueel gemarkeerd, zodat gebruikers hun voortgang kunnen zien.

De MySQL-database beheert de opslag van taken via phpMyAdmin.
